Добро пожаловать в репозиторий с материалами для проекта курса "Нейронауки и нейроинтерфейсы"

## Описание

Этот репозиторий содержит проектные занятия, примеры кода и документацию, разработанные в рамках курса для работы с нейроинтерфейсом Neiry HeadBand

Финальная цель проекта – практически реализовать работающий нейроинтерфейс для управления машинкой на ESP32 с использованием нейроинтерфейса Neiry Headband и библиотеки MNE-Python

## Структура репозитория

*   [`lesson_1`](./lesson_1): Подключение Neiry HeadBand и работа с EEG в реальном времени 
*   [`lesson_2`](./lesson_2): Фильтрация сигналов, выделение ритмов, базовый классификатор
*   [`lesson_3`](./lesson_3): Работа с ESP32 и передача команд на устройство
*   `lesson_4`: (Будет добавлено) Презентация финального проекта
*   [`CapsuleAPI`](./CapsuleAPI/CapsuleApp/CapsuleAppReleases): API система с готовой визуализацией ЭЭГ, PSD и метрик

## Начало работы

1.  Убедитесь, что у вас установлен Python 3.11+
2.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/GribanovDS/BSc-Neuroscience-and-Neurointerfaces-2024.git
    ```
3.  Перейдите в папку с нужным уроком (например, `lesson_1`)
4.  Установите зависимости, указанные в `requirements.txt` для соответствующего урока (если есть)
5.  Следуйте инструкциям, приведённым в `README.md` папки урока
